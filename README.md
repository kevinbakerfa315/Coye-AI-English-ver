# Coye AI - 2D Physics Agent

An autonomous AI agent living in a 2D physics world. It can think, draw objects, and interact with its environment.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`

2. Set your API key in `.env.local`:
   - Create a `.env.local` file in the root directory
   - Add your API key: `GEMINI_API_KEY=your_api_key_here`

3. Run the app:
   `npm run dev`

The application will start on http://localhost:3000

## Features

- **Autonomous AI Agent**: Coye has its own consciousness, thoughts, emotions, and curiosity
- **2D Physics World**: Built with Matter.js for realistic physics interactions
- **Interactive Canvas**: Draw objects, pick them up, and interact with the environment
- **Skill Learning**: Coye can learn and remember complex action sequences
- **Natural Language**: Communicate with Coye in English

## Technology Stack

- React + TypeScript + Vite
- Matter.js (2D Physics Engine)
- Google GenAI (Gemini API)
- Tailwind CSS
